### nvm 세팅법

- nvm Upgrade

  - ```
    (
      cd "$NVM_DIR"
      git fetch origin
      git checkout `git describe --abbrev=0 --tags --match "v[0-9]*" origin`
    ) && . "$NVM_DIR/nvm.sh"
    ```

- nvm install lts/* --reinstall-packages-from=node

- nvm alias default node
